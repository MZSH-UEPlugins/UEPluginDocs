# Simple Level Sequence Smooth

场景序列数据滤波处理插件，支持多种平滑算法对动画数据进行滤波处理。

## 功能特性

- **5 种平滑算法**：
  - **移动平均**：窗口内算术平均，快速预览去噪效果
  - **高斯平滑**：高斯分布加权平均，速度和加速度曲线也平滑，适合四肢大关节
  - **指数平滑**：指数加权递推，近期数据权重更大，适合实时控制预处理
  - **Butterworth 低通**：频域滤波，前后双向零相位无延迟，推荐用于关节数据
  - **Savitzky-Golay**：局部多项式拟合，精确保留峰值和拐点，适合手指等精细关节
- **内容浏览器集成**：右键 LevelSequence 资产即可使用滤波功能
- **ControlRig 支持**：直接处理 ControlRig 参数段（Transform 和 Scalar）
- **灵活输出**：可选覆盖原序列或创建新序列
- **批量处理**：一次处理多个序列，自动保持一致的后缀命名
- **自动帧率检测**：自动读取序列帧率用于 Butterworth 采样率

## 安装

1. 将 `SimpleLevelSeqSmooth` 文件夹复制到项目的 `Plugins/` 目录
2. 重启 Unreal 编辑器
3. 如未自动启用，在 编辑 > 插件 中启用

## 使用方法

1. 在内容浏览器中右键选择一个或多个 LevelSequence 资产
2. 从右键菜单中选择 **滤波**
3. 选择平滑算法并调整参数
4. 选择输出方式（覆盖原序列或创建新序列）
5. 点击 **确定** 应用

## 算法指南

| 算法 | 适用场景 | 关键参数 |
|------|----------|----------|
| 移动平均 | 快速预览 | 窗口大小 |
| 高斯平滑 | 四肢大关节 | 窗口大小、标准差 |
| 指数平滑 | 实时预处理 | 衰减系数 |
| Butterworth | 关节数据（推荐） | 截止频率 |
| Savitzky-Golay | 手指等精细关节 | 窗口大小、多项式阶数 |
